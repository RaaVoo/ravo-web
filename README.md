## 💡 Main Features

- 🎙 **음성 리포트 자동 생성** — 아이의 발화 데이터를 분석하여 감정 리포트 생성  
- 🎥 **영상 리포트 분석** — 표정과 행동 패턴을 기반으로 감정 분류  
- 🌤 **감정 날씨 시각화** — 긍정/부정 감정 비율을 감정 날씨로 표현  
- ☁️ **키워드 클라우드** — 주요 감정 단어 시각화  
- 👩‍👧 **부모용 리포트 페이지** — 아이 감정의 변화 추이를 한눈에 확인
